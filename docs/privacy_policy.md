# CyberLex Sweden Privacy Policy

## Purpose

This Privacy Policy explains how CyberLex Sweden handles information in the current prototype version.

CyberLex Sweden is an educational prototype created as a final school project.

---

## Current Prototype

The current version of CyberLex Sweden runs locally using Streamlit.

In the local prototype version, user questions are processed on the user's own machine while the app is running locally.

The current prototype does not include:

- user accounts
- login functionality
- payment processing
- database storage of user questions
- analytics tracking
- advertising tracking
- public user registration

---

## User Questions

Users can enter questions into the CyberLex Sweden interface.

The application uses the question to search local Markdown knowledge base files and generate a simple source-based answer.

In the current local prototype, CyberLex Sweden does not intentionally store user questions in a database.

---

## Local Files

CyberLex Sweden uses local Markdown files stored in the `data/` folder as its knowledge base.

The app reads these files to find relevant source sections.

---

## No Sensitive Data Intended

Users should not enter sensitive personal data, confidential legal information, secrets, passwords, access tokens, or private incident details into the prototype.

CyberLex Sweden is not designed to handle confidential legal matters or sensitive production data.

---

## Future Public Deployment

If CyberLex Sweden is deployed publicly in the future, this Privacy Policy must be reviewed and updated.

A public version may require clearer rules about:

- server logs
- user input storage
- analytics
- cookies
- third-party services
- hosting provider processing
- security measures
- retention periods
- user rights
- contact information

---

## Third-Party Services

The current local prototype uses Streamlit as the application framework.

If the project later connects to external services, such as a hosted language model, cloud database, analytics tool, or deployment provider, those services must be documented in an updated Privacy Policy.

---

## Data Security

The current prototype should not be used for confidential or sensitive legal matters.

If the project becomes public, additional security measures should be added before real users interact with it.

---

## Legal Disclaimer

CyberLex Sweden does not provide legal advice.

Users should verify important legal or compliance questions with official sources or qualified professionals.
