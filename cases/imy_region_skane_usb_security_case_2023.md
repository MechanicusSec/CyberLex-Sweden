# Region Skåne USB Security Case

## Case type

IMY decision / GDPR security measures / sensitive personal data / lost storage device

## Jurisdiction

Sweden

## Year

2023

## Authority or court

IMY, Integritetsskyddsmyndigheten

## Topic

GDPR, personal data breach, sensitive personal data, security measures, encryption, removable storage, healthcare data, administrative fine.

## Short summary

IMY issued an administrative fine of SEK 200,000 against Region Skåne after an unencrypted USB memory device containing sensitive personal data was lost.

The USB memory device contained personal identity numbers and sensitive healthcare-related personal data about 1,934 people.

The case concerned the protection of personal data stored on removable media, which is especially risky because such devices can easily be lost, stolen, copied, or accessed by unauthorized persons.

IMY assessed that Region Skåne had not ensured an appropriate level of security for the personal data on the USB memory device.

The issue was not a cyberattack, ransomware incident, malware infection, or external intrusion.

Instead, the risk came from physical loss of storage media combined with insufficient technical protection.

The case is important because it shows that GDPR security requirements apply to practical everyday handling of data, not only to servers, networks, and cloud systems.

It also shows why encryption is an important safeguard when sensitive personal data is stored on portable devices.

Healthcare-related information is especially sensitive because exposure can create serious risks for affected individuals.

The case is useful as an example of how weak removable-media controls can become a personal data breach and lead to an administrative fine.

## What happened

An unencrypted USB memory device containing personal data was lost.

The USB memory device contained personal identity numbers and sensitive healthcare-related personal data about 1,934 people.

The incident concerned removable storage, which means the data was stored on a portable device rather than only inside a controlled central system.

Because the device was not encrypted, the loss created a risk that unauthorized persons could access the personal data if they found or obtained the USB memory device.

The information involved healthcare-related data, which means that the incident concerned sensitive personal data under the GDPR.

IMY assessed the case as a security issue connected to the protection of personal data on removable storage media.

The case shows that organizations must consider the risk of loss, theft, or unauthorized access when personal data is stored on portable devices.

It also shows that removable media should not be treated as harmless simply because the device is small or used for practical everyday work.

The incident was not caused by a hacker breaking into a system, but by inadequate protection of data stored on a physical device.

## Legal issue

The main legal issue was whether Region Skåne had used appropriate technical and organisational measures to protect sensitive personal data stored on a USB memory device.

The case is relevant to GDPR security, confidentiality, encryption, removable media controls, healthcare data protection, and personal data breach handling.

It also raises the question of how organizations should control the use of portable devices when the data involves health information or other sensitive personal data.

The case connects directly to Article 32 GDPR, which requires appropriate security measures based on the risk of the processing.

## Decision or outcome

IMY issued an administrative fine of SEK 200,000 against Region Skåne.

IMY assessed that the personal data on the USB memory device had not been protected with an appropriate level of security.

The case shows that the loss of a portable storage device can become a GDPR issue when personal data is not protected with appropriate safeguards.

It also shows that encryption can be a central safeguard when sensitive personal data is stored on removable media.

The decision is a useful example of how physical handling of data can create GDPR risk even when there is no digital intrusion.

## Learning note

This case shows that portable storage devices should be treated as high-risk when they contain personal data, especially sensitive personal data.

Encryption, access controls, minimisation, clear routines, and secure storage practices are important safeguards.

Organizations should also consider whether removable storage is necessary at all, especially when the same work can be done in a controlled system with logging, access controls, and backup routines.

If removable media must be used, the organization should define rules for encryption, approved devices, storage, transport, deletion, incident reporting, and documentation.

The painfully simple lesson, because humans keep needing it carved into stone, is that small devices can create large GDPR problems.
